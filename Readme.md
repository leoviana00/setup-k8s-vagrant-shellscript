## Project

- Setup kubernetes com shell script, vagrant e virtualbox.

## Pré-requisitos

- vagrant
- virtualbox

## Execução
1. Instalação dos pré-requisitos:

```bash
make requirements
```

2. Subir ambiente

```bash
make setup
```

3. Remover o ambiente

```bash
make destroy -f
```

4. Parar o ambiente

```bash
make halt
```
